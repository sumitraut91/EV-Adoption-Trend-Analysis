# 🚗 Driving Change: EV Adoption Trends Across the U.S.

## 📘 Overview

This project presents an **interactive Excel dashboard** analyzing electric vehicle (EV) adoption trends across the United States. It focuses on understanding the distribution and growth of **Battery Electric Vehicles (BEVs)** and **Plug-in Hybrid Electric Vehicles (PHEVs)**, along with the impact of **CAFV (Clean Alternative Fuel Vehicle) eligibility**.

The analysis is built using real-world EV registration data and showcases how Excel can be used as a powerful data analytics and visualization tool.

---

## 🎯 Objectives

* Analyze EV adoption trends across U.S. states
* Compare BEVs and PHEVs by volume and performance
* Evaluate CAFV eligibility distribution
* Identify leading manufacturers and models
* Build an interactive and user-friendly dashboard

---

## 📊 Dashboard Features

* **KPI Cards**: Total EVs, BEVs, PHEVs with percentage breakdown
* **Map Visualization**: State-wise EV distribution
* **Trend Analysis**: EV registrations by model year
* **CAFV Eligibility Pie Chart**
* **Top 10 Manufacturers Chart**
* **Electric Range Treemap**
* **Interactive Filters (Slicers)**:

  * EV Type
  * CAFV Eligibility
  * Model Year
  * Make & Model

---

## 🗂️ Dataset Information

* **Source**: Kaggle EV Population Dataset
* **Records**: 177,000+ entries
* **Key Columns**:

  * Vehicle ID (VIN partial)
  * State, City, ZIP
  * Make, Model, Model Year
  * EV Type (BEV / PHEV)
  * Electric Range
  * CAFV Eligibility

---

## 🧹 Data Cleaning & Preparation

Performed using **Power Query**:

* Removed unnecessary columns (MSRP, utility, etc.)
* Standardized data types
* Handled missing/invalid values
* Created a structured **Cleaned Data** sheet

## 🔍 Key Insights

* **BEVs dominate** the EV market (~79%)
* **Tesla leads** in both registrations and electric range
* **PHEVs often fall short** of CAFV eligibility criteria
* **Washington State** shows highest EV adoption 177477
* Adoption is influenced by **policy, infrastructure, and performance**
